Themeing

This library is designed so that all of the element styles are informed by the values in a single theme file.

The default theme included provides a fairly generic visual design language as a starting point — but a completely new visual design can be achieved by creating a new theme.

Default Theme Configuration

These are the variables and configurations for the default theme. Create your own theme by changing these values in a new theme partial.
