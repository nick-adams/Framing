Centered Columns

Add '.grid__col--centered' to any column except '.grid__col--onewhole' (centering a one whole column is unnecessary).

.grid__col--onewhole
.grid__col--threequarters
.grid__col--centered
.grid__col--twothirds
.grid__col--centered
.grid__col--onehalf
.grid__col--centered
.grid__col--onethird
.grid__col--centered
.grid__col--onequarter
.grid__col--centered

Content Alignment Options

Add '.grid__col--contentright' or '.grid__col--contentcenter'. There is no '.grid__col--contentleft', since content left aligned is the default.

.grid__col--onehalf
(content left aligned is default)
.grid__col--onehalf
.grid__col--indent-onehalf
.grid__col--contentright
.grid__col--onethird
.grid__col--indent-twothirds
